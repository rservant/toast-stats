# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, report it privately by one of the following methods:

- GitHub Security Advisories (preferred)

We will acknowledge receipt of your report within 72 hours and work with you to assess and remediate the issue.

## Supported Versions

Only the latest released version of this project is currently supported with security updates.
